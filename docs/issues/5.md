# Issue #5: 「ようこそ、Test Page」の下に文言を配置

## 対応プラン

Issue内容：「ようこそ、Test Page」の下に「一緒の挑戦しましょう」という文言を配置する。

実装方針：
1. `app/page.tsx`のメインコンテンツ部分を修正
2. h1要素とp要素をdivでラップして、両方を中央揃えに保持
3. 新しいp要素で「一緒の挑戦しましょう」を追加
4. 既存のスタイリング（白文字、ドロップシャドウ）と統一感を持たせる

## 対応内容

### 変更ファイル
- `app/page.tsx`

### 実施した修正
1. メインコンテンツのh1要素を`<div className="text-center">`でラップ
2. h1要素の下に以下のp要素を追加：
   ```tsx
   <p className="text-2xl text-white mt-4 drop-shadow-lg">
     一緒の挑戦しましょう
   </p>
   ```

### 結果
「ようこそ、Test Page」の下に「一緒の挑戦しましょう」というテキストが表示されるようになった。両方のテキストは中央揃えで、統一されたスタイリングが適用されている。
